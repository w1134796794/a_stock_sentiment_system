# A股短线情绪量化系统

基于龙头战法、弱转强、板块切换逻辑的短线交易辅助系统。

## 🎯 核心功能

### 1. 数据层 (Data Layer)
- **双源备份**: Tushare(历史数据) + AkShare(实时情绪)
- **本地缓存**: 自动避免重复API调用，支持盘中/盘后增量更新
- **智能交易日判断**: 非交易日自动关联最近交易日数据
- **东财行业板块数据**: 集成东方财富行业板块数据，包含涨跌比、领涨强度等因子

### 2. 分析引擎 (Core Engine)
- **层级映射**: L1(一级行业) -> L2(二级行业) -> L3(三级行业) -> 核心标的
  - 使用 `data/Industry_Mapping.csv` 作为行业映射源
  - 31个一级行业，127个二级行业，338个三级行业
- **主线强度**: 综合权重 = 20日涨停数×0.35 + 当日涨停数×0.35 + 最高连板×0.15 + 东财综合强度×0.15
- **梯度追踪**: 1B/2B/3B/4B/5B/6B+ 分布统计，板块联动判定
- **情绪指标**: 炸板率、昨日涨停溢价、市场温度

### 3. 模式识别 (Pattern Recognition)
- **弱转强**: 昨日烂板/炸板 + 今日跳空高开 > 2% + 今日涨停
- **二板定龙**: 昨日首板 + 今日高开3-7% + 15分钟内涨停
- **首板突破**: 早盘秒封(9:40前) + 封单强度>5% + 换手5-20%
- **分歧转一致**: 昨日烂板爆量 + 今日高开2-5% + 30分钟内涨停
- **卡位板**: 同板块内低位股抢先涨停5分钟以上

### 4. 市场主线分析 (Market Mainline)
- **20日涨停统计**: 按三级行业统计涨停次数、涉及股票数
- **主线板块识别**: TOP5主线板块，包含层级信息
- **关注标的筛选**: 出现2次以上 + 最高连板≥2 + 平均涨幅≥9.5%
- **东财因子融合**: 涨跌比、领涨强度、活跃度等因子融入主线分析

### 5. 核心标的概念数据
- **概念成分股数据**: 每日获取东财概念板块成分股数据（dc_member接口）
- **自动概念匹配**: 为核心标的自动匹配所属概念
- **概念去重**: 自动去重并用逗号分隔
- **本地缓存**: 概念数据每日缓存，避免重复API调用

### 6. 可视化报告 (Reporting)
- **Excel多Sheet报表**:
  - Dashboard: 市场情绪、主线Top5、涨停梯队
  - 主线板块Top5: 包含东财因子数据（涨跌比、领涨强度等）
  - 板块热度分析: 3日/5日/20日热度统计
  - 涨停梯队: 按连板高度分组
  - 模式信号: 交易机会列表
  - 核心标的: 按L1-L2-L3层级分组，包含所属概念
- **Markdown分析报告**: 市场主线分析报告

## 🛠️ 安装与配置

### 步骤1: 安装依赖
```bash
cd a_stock_sentiment_system
pip install -r requirements.txt
```

### 步骤2: 配置API Token
编辑 `config/settings.py`:
```python
TUSHARE_TOKEN = "你的tushare_token_here"
```

### 步骤3: 准备行业映射
系统使用 `data/Industry_Mapping.csv` 作为行业映射源，格式如下：
```csv
一级行业,二级行业,三级行业
电力设备,光伏设备,光伏组件
电力设备,光伏设备,光伏辅材
...
```

## 🧠 交易逻辑整合

### 数据落库 (15:40)
系统自动抓取今日数据，存入 `data/raw/YYYYMMDD/`

### 决策流程
1. **看梯度**: 今天市场最高板是多少？断层了吗？
2. **看主线**: 哪些L3板块涨停数在递增？
3. **看因子**: 东财行业板块数据（涨跌比、领涨强度）
4. **盯核心**: 在这些主线L3中，谁是"弱转强"？谁是"卡位板"？
5. **看概念**: 核心标的所属概念是否热点？

### 制定计划
从Excel的Watchlist中选出3-5只符合逻辑的个股，作为次日竞价关注标的。

## 🔧 核心模块说明

### data_manager.py
- **东财行业板块数据**: `get_industry_sector_data()` - 获取行业板块数据，计算涨跌比、领涨强度等因子
- **概念成分股数据**: `get_concept_members()` - 获取所有概念板块的成分股，每日缓存
- **个股概念匹配**: `get_stock_concepts_from_members()` - 从成分股数据中匹配个股所属概念
- **核心标的概念**: `enrich_core_stocks_concepts()` - 批量为核心标的添加概念数据

### industry_mapper.py
- 加载 `data/Industry_Mapping.csv`
- 提供 L1/L2/L3 层级映射查询
- 支持根据三级行业查找上级行业

### pattern_recognition.py
- 检测5种交易模式
- 基于涨停池数据进行模式匹配
- 输出带置信度的交易信号

### sector_heat_calculator.py
- 计算3日/5日/20日板块热度
- 分析板块动量加速度
- 识别新主线、老主线、退潮板块

### analyze_market_mainline.py
- 采集最近20个交易日涨停数据
- 按三级行业统计涨停次数
- 识别主线板块和关注标的

## 📈 数据流程

```
1. 数据采集 (AkShare/Tushare)
   ↓
2. 东财数据获取 (dc_index/dc_member)
   ↓
3. 行业映射 (Industry_Mapping.csv)
   ↓
4. 模式识别 (5种模式)
   ↓
5. 主线分析 (20日统计 + 东财因子)
   ↓
6. 概念匹配 (dc_member数据)
   ↓
7. 报告生成 (Excel + Markdown)
```
