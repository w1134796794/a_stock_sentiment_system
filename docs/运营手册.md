# A股情绪系统 · 短视频 / 小红书 合规运营手册

> 一句话总纲：**讲方法不讲标的，讲过程不讲收益，做工具不做决策，每条带风险提示，私信不给建议。**
>
> ⚠️ 本手册为运营思路整理，非法律意见。证券类内容监管很重，正式推广前请让法务/合规过一遍。

---

## 0. 合规红线（决定一切的前提）

无「证券投资咨询」资质时，全程**不得**：

- 推荐具体个股 / 给买卖点 / 目标价 / "明天买这只"
- 承诺或暗示收益、"稳赚 / 保本 / 翻倍 / 胜率90% / 财富自由"
- 晒收益截图诱导转化、晒"抓到的涨停"做钩子
- 诱导交易（"赶紧上车 / 错过后悔"）
- 出现敏感词：牛股、抓涨停、内幕、必涨、稳了、跟我买

平台层面：抖音 / 小红书对证券内容限流极严，**每条内容口播 + 字幕 + 文案三处都要带风险提示**。

## 1. 产品定位（三选一或叠加）

| 定位 | 卖点 | 风险 |
|---|---|---|
| 效率工具向（最稳） | 把盘后复盘从 2 小时压到 10 分钟，决策永远是用户自己做 | 最低 |
| 投教科普向（涨粉最快） | 把情绪周期 / 龙头梯队 / 量价封单拆成知识点 | 低（讲方法不讲标的） |
| 技术 / 独立开发者向（零监管） | "我如何用 Python 搭一套 A 股情绪看板"，走技术分享 | 几乎无 |

主推：**投教（公域涨粉）+ 效率工具（产品转化）**，技术向做差异化。

## 2. 十条可直接拍的脚本 / 选题

> 每条结尾固定风险提示；不点名个股、不报收益、不给买卖点。

1. **痛点开场**：盘后手工记涨停梯队记到凌晨——录屏切到梯队卡片，强调省时间。
2. **科普·情绪四阶段**：冰点→上升→高潮→退潮，4 张卡片讲特征。
3. **科普·什么是龙头梯队**：一张图讲连板高度/家数，讲结构不讲哪只。
4. **方法论·满分扣分制 vs 加分制**：用扣分明细卡片讲评估思路（打码股名）。
5. **工具展示·10秒看完全市场情绪**：录屏划过仪表盘，只报整体状态。
6. **幕后·打工人自建系统**（最圈粉）：为什么做、踩了什么坑、现在长啥样。
7. **科普·情绪周期决定胜率**：胜率热力表，标注"历史统计、不代表未来"。
8. **工具向·我的盘后复盘 SOP**：大盘情绪→梯队→资金→记笔记。
9. **误区纠偏·别只盯涨停**：看"涨停的结构"而非单只。
10. **互动·情绪温度你打几分**：抛当日整体情绪状态，引导评论。

## 3. "情绪温度计"日更模板

> 定位：**每天只播报市场整体情绪状态**，绝不点名个股 / 不预测 / 不给操作。养号 + 守合规利器。
> 代码已实现自动生成（见第 5 节），字段直接来自看板快照。

**口播脚本模板（30 秒，填变量）**

```
【月-日】A股情绪温度计。
今天市场处于【冰点 / 上升 / 高潮 / 退潮 / 震荡】期。
全场涨停【N】家、跌停【M】家，最高连板【B】板，炸板率【R】%。
涨停梯队呈「1板X家·2板Y家·3板Z家」，市场宽度上涨【U】家、下跌【D】家。
热点集中在【概念1、概念2】方向。
一句话总结：今天情绪【偏暖 / 转冷 / 高位分歧 / 修复中】。
以上为市场状态记录，不构成投资建议，股市有风险。
```

**小红书图文版（4 图）**：封面（情绪温度大字）→ 情绪周期 → 涨停梯队结构 → 主线方向 + 总结；正文 ≤200 字，结尾固定风险提示；标签 `#复盘笔记 #交易日记 #市场观察`（**避开** `#荐股 #牛股`）。

## 4. 合规话术 & 敏感词清单

### 4.1 固定风险提示（四场景复用）

- **口播版**：以上为工具演示/市场记录，不构成投资建议，股市有风险，决策请独立判断。
- **字幕版**：内容仅供学习交流，不构成投资建议 · 市场有风险
- **图文版**：📌 本内容为个人复盘/工具记录，不构成任何投资建议，据此操作风险自负。
- **主页简介版**：分享复盘方法与自用效率工具｜不荐股·不代客理财·不构成投资建议

### 4.2 能说 ✅ / 不能说 ❌（替换对照）

- ❌ 这只票明天能涨 → ✅ 今天市场整体处于 X 情绪阶段
- ❌ 建议买入 / 上车 / 止盈 → ✅ 我自己的复盘流程是……（决策你自己定）
- ❌ 胜率90% / 稳赚 / 翻倍 → ✅ 这是历史统计口径，不代表未来
- ❌ 抓涨停 / 牛股 / 必涨 → ✅ 看涨停梯队的结构变化
- ❌ 跟我操作 / 带你飞 → ✅ 分享一个看市场的视角
- ❌ 内幕 / 消息面利好 X 股 → ✅（不碰，直接删）
- ❌ 晒收益截图引流 → ✅ 展示界面 / 方法 / 流程，不展示盈亏

### 4.3 敏感词黑名单

- **荐股类**：荐股、牛股、抓涨停、买入、卖出、目标价、上车、抄底、加仓、清仓
- **收益类**：稳赚、保本、包赚、翻倍、暴利、躺赚、月入X万、财富自由、收益率XX%
- **诱导类**：跟我买、闭眼入、错过后悔、最后机会、赶紧、必涨、稳了
- **夸大类**：100%、绝对、保证、唯一、最强
- **资质类**：内幕、消息、机构票、游资合作、代客理财、保证收益

### 4.4 账号 & 互动红线

- 简介/昵称不含"荐股、操盘、带单、收益"。
- 私信/评论区**不一对一给个股或买卖建议**（一对一更易被认定咨询行为）。
- 不接荐股广告、不做付费"喊单"社群。
- 转化只导向"工具试用 / 方法论付费 / 知识社群"，反复声明不提供投资建议。
- 历史数据演示，个股名称/代码打码或用脱敏数据。

## 5. 一键生成图文 / 视频（已用代码实现）

模块：`recap/thermometer.py`（合规版「情绪温度计」生成器，复用现有 storyboard/视频管线，但**只输出市场整体聚合数据 + 合规文案**，自动剔除个股名/选股/仓位建议）。

### 用法

```bash
# 生成最新交易日的 文案 + 图文卡片HTML（默认）
python -m recap.thermometer

# 指定交易日
python -m recap.thermometer 20260611

# 同时渲染视频（需 Node.js≥22 + npx hyperframes；缺失则只打印命令）
python -m recap.thermometer 20260611 --video
```

### 产物（写入 `webdata/content/{date}/`）

- `文案.md`：口播稿 + 小红书标题/正文 + 标签 + 风险提示，复制即用。
- `卡片.html`：自包含 9:16 暗色卡片页，浏览器打开点「下载全部PNG」即得小红书图。
- `story.json`：合规分镜（recap.v1 schema），可走视频管线出 MP4。

### 与原管线的区别（关键）

原 `recap/storyboard.py` 会**点名个股、显示选股/置信度/明日仓位** → 适合自己看，**不适合公开发布**。
`thermometer.py` 是其**合规子集**：只描述"已发生的市场整体状态"，不做个体预测、不给操作建议。对外发布一律用 thermometer。
