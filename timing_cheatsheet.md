
# 各模式介入时机与仓位管理速查表

| 模式 | 介入时机 | 目标价格 | 仓位 | 次日计划生成 |
|------|---------|---------|------|-------------|
| **二板定龙** | 竞价末段(9:24:30) > 开盘(9:30) | 涨停价 | medium | ✅ 是 |
| **分歧转一致** | 竞价末段 > 开盘第一笔 | 开盘价 | medium | ✅ 是 |
| **弱转强** | 竞价末段 > 开盘第一笔 | 开盘价 | **heavy** | ✅ 是 |
| **首板突破** | 涨停瞬间(9:40前) | 涨停价 | light | ✅ 是 |
| **竞价爆量** | 竞价末段(9:24:30) | 竞价匹配价 | medium | ✅ 是 |
| **炸板回封** | **次日**竞价/开盘 | 开盘价 | light | ✅ 是(次日执行) |
| **卡位板** | 涨停瞬间(早盘) | 涨停价 | medium | ⚠️ 盘中识别 |
| **龙二波** | 启动日首板/次日竞价 | 涨停价/开盘价 | medium | ✅ 是 |
| **龙回头** | 回踩均线时(早盘/下午) | 均线价 | light | ✅ 是 |

## 复盘后自动生成计划的流程

```
15:40 收盘后
  ↓
1. 获取当日所有数据（涨停池、行情、板块热度）
  ↓
2. 运行所有策略检测（二板、弱转强、炸板回封等）
  ↓
3. 生成信号 → 转换为交易计划（介入时机、价格、仓位）
  ↓
4. 按介入时间排序，生成CSV文件
  ↓
5. 发送交易报告（微信/邮件）
  ↓
次日09:15 开盘前
  ↓
6. 加载交易计划，准备执行
  ↓
09:25-15:00 实时监控
  ↓
7. 到达介入时机 → 检查条件 → 触发交易/取消
```

## 关键原则

1. **竞价定生死**：二板定龙、弱转强、分歧转一致，都在竞价末段决定
2. **开盘定强弱**：9:30-9:35确认是否执行，不犹豫
3. **打板是最后选择**：多数模式买点在涨停前，打板已晚
4. **次日执行模式**：炸板回封必须次日竞价确认，当日不追
5. **取消条件优先**：任一取消条件满足，立即放弃，不心存侥幸
